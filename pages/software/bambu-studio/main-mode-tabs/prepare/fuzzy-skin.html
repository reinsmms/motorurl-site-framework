Prepare → Fuzzy Skin

Fuzzy skin adds intentional surface texture by applying small, randomized outer-wall offsets. It can improve appearance and help coatings (like Sno Tex) “grab” the surface — but it can also increase vibration on thin parts.

What It Is

Fuzzy skin changes the toolpath so the nozzle makes many small direction changes along the outer wall. The result is a roughened surface that looks “cast” or “weathered.”

Controlling Density

  • Fuzzy skin point distance: smaller distance = denser fuzz.
  • Fuzzy skin thickness: larger thickness = deeper texture but more lateral force.

Tip: Adjust point distance first to increase texture density. Increasing thickness often causes instability before improving appearance.

Fuzzy Skin Painting Tool

Allows fuzzy skin to be applied selectively instead of across the entire model. Useful for roofs, decorative faces, or specific texture areas.

Reducing Dynamic Excitation (Wall Vibration)

Rapid direction changes repeatedly push sideways on the wall. Thin, tall, and on-edge parts amplify this effect.

Stability Adjustment Priority

  1. Increase point distance — Suggested: 0.45 – 0.6 mm
  2. Reduce thickness — Suggested ceiling: ≤ 0.2 mm
  3. Lower outer wall speed — Suggested: 40 – 60 mm/s
  4. Lower outer wall acceleration — Suggested: 800 – 1500 mm/s²

Safe Starting Preset (Thin Vertical Parts)

  • Thickness: 0.2
  • Point distance: 0.5
  • Outer wall speed: 50 mm/s
  • Outer wall acceleration: 1000 mm/s²
