Main Mode Tabs → Prepare

The Prepare tab is where you bring a model in, orient it, set print + filament profiles, apply modifiers, supports, and per-object settings, and get everything ready before slicing.

Key Areas in Prepare

  • Model placement: move, rotate, scale, mirror, arrange.
  • Print / Filament settings: quality, strength, speed, temperatures, cooling.
  • Per-object settings: modify settings for a specific part or region.
  • Supports: generate and tune supports before slicing.
  • Surface / appearance tools: texture and finish features like Fuzzy Skin.
